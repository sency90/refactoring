# 문자열 유사도 검사 프로그램

두 문자열의 유사도를 검사하는 프로그램입니다.  
유사도의 결과는 **0~100점**으로 표기됩니다.

---

## 1. 글자 수 검사 (최대 60점)

- 두 문자열의 **글자 수가 같을 경우**: **60점**
- **문자열 길이 차이가 2배 이상일 경우**: **0점**
- 그 외는 아래 공식을 사용하여 부분 점수 계산

### 점수 계산 공식

```
A = 긴 문자열 길이  
B = 짧은 문자열 길이

부분 점수 = (1 - (A - B) / B) × 60
```

---

## 2. 등장 알파벳 검사 (최대 40점)

- 두 문자열에 **동일한 종류의 알파벳**이 사용된 경우: **최대 40점**
- 사용된 알파벳이 **모두 다르면**: **0점**
- 비교 시 **대문자만 취급**

### 점수 계산 공식

```
TotalCnt = 두 문자열에서 사용된 알파벳의 총 종류 수  
SameCnt = 두 문자열 모두에 공통으로 사용된 알파벳 종류 수

부분 점수 = (SameCnt / TotalCnt) × 40
```

---

## 최종 점수

최종 유사도 점수는 **글자 수 검사 점수 + 알파벳 검사 점수**를 합산하여  
**0점 ~ 100점** 사이의 정수로 계산됩니다.
